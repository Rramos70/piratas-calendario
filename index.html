📌 ¿Cómo funciona?
Jugadores: Solo necesitas configurar el repositorio (Owner + Repo) para ver el calendario.

Administrador: Además del repositorio, añade un Token de GitHub para poder crear, editar y eliminar partidos.

Pasos para el Token:
1. GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic) → "Generate new token"
2. Marca el scope repo y genera
3. Copia el token (empieza con ghp_)
